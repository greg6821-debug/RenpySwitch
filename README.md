## Описание

Данный репозиторий служит сборщиком RenPy на Github Actions

Загружает исходники RenPy

Применяет патчи к RenPy

Компилирует RenPy

Пакует RenPy в lib.zip

Собирает пустой проект с .bat файлом для сборки игры с версией 7.6.1.

Собраный файл с игрой работает только на Свиче, на эмуляторе новеллы не запускаются - сразу вылетают.
